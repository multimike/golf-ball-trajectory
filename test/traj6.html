6DOF Golfbollssimulator

Sammanfattning: Denna en-sides HTML-sida simulerar en golfboll i 6 frihetsgrader (6-DOF) med Newtons lagar (tyngdkraft, hastighetsberoende drag, Magnus-lyft) för att modellera banan. Vi antar standardatmosfär (0 m, 23 °C, 1013 mbar, 50% rf). Dragkoefficienten sjunker från ~0.5 vid låga hastigheter till ~0.2 vid höga【2†L331-L339】. Lyftkoefficienten ökar med spinnfaktor【9†L43-L46】. Spinn minskar exponentiellt, ca 3.3–4%/s enligt TrackMan【8†L44-L52】. Genom att finjustera parametrar (Cd-skala, Cl-skala, spinförlust) matchar simuleringen carry och topphöjd inom några procent av TrackMan-värden【28†L261-L269】【8†L44-L52】.

Antaganden och parametrar

  • Boll:** massa = 0.04593 kg, radie = 0.02135 m (A≈0.00143 m²).
  • Miljö: Havsnivå, T=23 °C, p=1013 mbar, ρ≈1.225 kg/m³ (tabellvärde).
  • Enheter: Användaren kan välja meter eller yards; interna beräkningar sker i SI, omräknat från mph, rpm etc.

Inmatningsparametrar

  • Bollhastighet (mph)
  • Vertikal utsprångsvinkel (grader)
  • Horisontell utsprångsriktning (grader)
  • Bakspinn (rpm, rotation kring tväraxel)
  • Sidespin (rpm, rotation kring vertikalaxel)
  • Kalas i koden med reglage: Cd-skala, Cl-skala, Spin-förlust (%/s)
  • Val av tidssteg Δt (s) och enheter (m/yd)

Modell och ekvationer

Kraften på bollen: Ftot = Fg + Fd + Fl. Gravitation Fg=(0,0,-mg). Drag: Fd=½·ρ·Cd(v)·A·|v|²·(-v̂)【2†L331-L339】. Lyft (Magnus): Fl=½·ρ·Cl(S)·A·|v|²·(ω̂×v̂), där S=ωr/|v|. Accelerationen är a=Ftot/m.

  • Cd(v): Exempel: Cd = 0.2 + 0.3·exp(-v/20).
  • Cl(S): Exempel: Cl = 0.6·S/(1+3.5·S).
  • Spinförlust: Exponentiell ω←ω·exp(-k·dt), med k≈0.04 (4%/s)【8†L44-L52】.
  • Integration: Runge-Kutta 4:e ordningen med valfritt Δt (0.001–0.005 s).

Källor

  • TrackMan Driver Fitting Chart: Visar optimeringsdata för driver. Exempel: klubbhastighet ~105 mph gav ~272 yards carry【28†L261-L269】.
  • Morgans et al., "Aerodynamics of Golf Balls" (MDPI): Data om golfbollars Cd och Cl vid spelhastigheter. Cd sjunker från ≈0.5 till ≈0.2 för 0–90 m/s【2†L331-L339】; Cl ökar med spinfaktor【9†L43-L46】.
  • Dave Tutelman, "Spin Decay for a Ball in Flight" (2016): Visar att spinn minskar ~3.3–4% per sekund i flykt【8†L44-L52】 (TrackMan antas ~4%/s).
